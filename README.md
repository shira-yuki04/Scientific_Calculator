# Scientific Calculator

A scientific calculator built with C# Windows Forms.

## Features
- Basic operations: +, -, *, /
- Power: x², x³, xⁿ
- Roots: √x, ∛x, ʸ√x
- Trigonometry: sin, cos, tan, cot, sec, cosec
- Logarithm: log, ln
- Factorial: x!
- Absolute value: |x|
- Percentage: x% of y
- Reciprocal: 1/x
- Backspace button
## How to Run
1. Clone the repo
2. Open `scientific_cal.sln` in Visual Studio
3. Press F5 to run
